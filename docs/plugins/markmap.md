# Markmap Plugin

`mkdocs-markmap` 插件允许你在 Markdown 文档中直接渲染思维导图（Mind Map）。它基于 markmap-lib，可以将简单的 Markdown 列表转换为交互式的思维导图。

## 使用方法

在 Markdown 文件中，使用带有 `markmap` 类的代码块：

```plaintext
# Root
## Branch 1
- Leaf 1.1
- Leaf 1.2
## Branch 2
- Leaf 2.1
```

## 效果演示

```markmap
# 思维导图示例
## 核心功能
- 自动布局
- 交互缩放
- 节点折叠
## 优势
- 简单易用
- 纯文本维护
```
