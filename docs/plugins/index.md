# MkDocs 插件列表

本博客使用了多个 MkDocs 插件来增强功能和体验。以下是已安装插件的列表及介绍：

## 导航与结构
- [Awesome Pages](awesome-pages.md): 强大的导航自定义插件，支持通过 `.pages` 文件控制目录结构。
- [Meta Descriptions](meta-descriptions.md): 自动从页面内容生成 meta description 标签，优化 SEO。

## 内容增强
- [Markmap](markmap.md): 支持在 Markdown 中直接渲染思维导图。
- [Mermaid2](mermaid.md): 支持渲染 Mermaid 图表（流程图、时序图等）。
- [Code Include](codeinclude.md): 支持从外部文件导入代码片段。
- [Include Markdown](include-markdown.md): 支持在文档中包含其他 Markdown 文件。
- [Copy Button](copy-button.md): 代码块一键复制功能（MkDocs Material 内置）。

## Git 信息
- [Git Revision Date Localized](git-revision-date.md): 显示页面最后更新时间。
- [Git Authors](git-authors.md): 显示页面贡献者/作者。

## 构建优化
- [Minify](minify.md): 压缩 HTML、JS 和 CSS 文件，减小体积。

## 其他
- [Static i18n](i18n.md): 静态国际化支持。
