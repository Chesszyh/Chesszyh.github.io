# Formal Languages and Formal Automata (FLFA，形式语言与形式自动机)

本门课程为同济信息安全大三学期专业必修课。

## 评分标准

1. 考试（50%）：期末闭卷考试50分；
2. 作业（30%）：共5次作业，每次6分；
3. 平时（10%）：平时综合表现；
4. 出勤（10%）：随机点名5次，每次2分；

## 期末考试

考试时间120分钟。

- 填空10x2
    - 计算理论：几种计算模型
    - 形式语言与自动机发展历史
    - 正则表达式编写
    - 邱奇-图灵论题
    - 可计算性：NP-C, 归约
- 大题4x20
    - 设计DFA(字母表0/1/2, 排除00/11/22子串)
    - 设计正则(字母表0/1, 排除00子串)
    - 证明某语言非正则
    - L(G) = {0^m 1^m 0^n 1^n | m,n>=0}，设计PDA和CFG

有人30分钟内做完交卷，有人30分钟时不会的题一小时之后还是不会。

我是后者。
