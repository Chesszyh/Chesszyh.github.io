# Software Engineering

本门课程为同济信息安全大三学期专业选修课，授课教师为Zhao Junqiao教授。

该门课程平时作业占20%，课程项目占50%，期末考试占30%。

平时作业共5次，到考前会发答案，题目每年估计不会怎么变。

课程项目为小组项目，分组后每组7人，与其他小组共同扮演甲乙方完成项目。我们的项目：[lovehome](https://gitlab.com/tj-cs-swe/CS10102302-2025/group2/lovehome)，基于kotlin开发的Android聊天应用，基本100%AI。

期末考前老师在canvas上发了一个79个单选的小测，考完发现基本没有原题。考试时长1h35min（两节课），题量稍大。

2026年1月题型：

- 10x1判断
- 10x1单选
- 8道简答，每题4分
- 4道设计题
    - 程序流程图、基本路径覆盖
    - 需求分析、SRS文档目录设计
    - UML类图设计
    - 程序代码段优化（2选1选做题，另外一道忘记了，没细看）
