# Better Vibe Coding

## 不负责任的Vibe Coding

最好通过只读用户登录，或者在容器中运行，以防止意外的数据丢失或泄露。

```bash
claude --dangerously-skip-permissions

codex --sandbox workspace-write --ask-for-approval never

gemini -m gemini-3-flash-preview  --yolo
```
